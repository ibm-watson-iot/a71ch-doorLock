Watson IoT Platform

IBM Watson IoT Platform service is used to define device types, connect devices securely, and manage the flow of device data. You need to configure IBM Watson IoT Platfor service to connect NXP A71CH Ready for Watson IoT devices and manage device data flow.

1. Register NXP Certificate Authority

NXP A71CH Ready for Watson IoT is an A71CH device that has been trust provisioned by NXP with credentials (one EC key pair, one Device X509 certificate and one Gateway X509 certificate) supporting TLS access to Watson IoT. For these devices to securely connect with Watson IoT Platform service, you need to register NXP Certificate Authority with Watson IoT Platform service.

2. Configure Connection Security Policy

NXP A71CH devices to connect using TLS, set Default Connection Security to TLS with Client Certificate Authentication.

3. Register Device Types

NXP A71CH Watson IoT type devices are trust provisioned with a Device X509 certificate and a Gateway X509 certificate. If you want to configure and connect a device, register NXP-A71CH-D as a generic device type with Watson IoT Platform service. If you want to configure and connect a gateway, register NXP-A71CH-G as a generic gateway type.
